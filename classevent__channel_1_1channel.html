Detailed Description

template<class DispatchPolicy = dispatch_policy::sequential, bool IdlePolicy = idle_policy::keep_events>
class event_channel::channel< DispatchPolicy, IdlePolicy >

The event channel. Handles subscriptions and message dispatching.

Template Parameters
DispatchPolicyHow to dispatch events. A type from dispatch_policy.
IdlePolicyWhat to do with incoming events when idle. A value from idle_policy.
